Command Reference

New Admin Commands

All commands added by Kolat. Arguments in <brackets> are required, [brackets] are optional.

Command Arguments Permission Description
/kolat — ALL Show mod info, version, and loaded modules.
/invsee <player> ADMIN Open and edit another player's inventory in real-time.
/freeze <player> [duration] ADMIN Completely immobilize a player. Optional duration in seconds.
/heal [player] [amount] OP Restore health to full or to a specified amount for self or target.
/feed [player] OP Instantly fill the hunger bar of self or another player.
/god [player] ADMIN Toggle godmode — invincibility to all damage types.
/vanish [player] ADMIN Make yourself or a player invisible to other players. Silent join/leave.
/broadcast <message> OP Send a formatted announcement to the entire server.
/tphere <player> OP Teleport a player to your current location instantly.
/smite <player|entity> ADMIN Strike a target with a powerful lightning bolt.
/world <time|weather|pvp> <value> ADMIN Control per-world settings including time cycle, weather, and PvP toggle.
/spawner <entity> ADMIN Place a custom mob spawner with the entity type of your choice.
/give custom <foodid> [amount] [player] OP Give any Kolat custom food item directly to a player.
/revive <player> ADMIN Restore a recently deceased player at their death coordinates with full HP.
/clearinv <player> [slot] ADMIN Clear all or specific slots of a player's inventory.

Installation Guide

Set up in 3 steps

01

Install Minecraft Forge

Make sure you have Minecraft Forge installed for your Minecraft version (1.20+ recommended). Download it from the official Forge website and run the installer. Launch the game once to generate the mods/ folder.

02

Download Kolat-5.0.2.jar

Click the download button below to get Kolat-5.0.2.jar from our official distribution. Save the file anywhere on your computer — you'll move it in the next step.

03

Place in your mods folder

Move Kolat-5.0.2.jar into your Minecraft mods/ folder. On Windows: %appdata%\.minecraft\mods\ — On Mac: ~/Library/Application Support/minecraft/mods/. Launch the game with the Forge profile and enjoy Kolat!
